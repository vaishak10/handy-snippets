# Sort Array 

<u>**Aim**</u>: Sort the array elements and print the array in ascending order.<br>

>**Input**: '[ 4, 9, 2, 3, 5, 1 ]'
>
>**Output**: '[ 1, 2, 3, 4, 5, 9 ]'

**Solution**:
```
let array = [4,9,2,3,5,1];
console.log(array.sort((a,b) => a-b));
```
